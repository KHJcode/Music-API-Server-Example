# Music API Server Example

## 📚 Description

This is an example of a music API server with DB connection (Node.js, Express, Sequelize).

<br/>

## 🛠️ Installation

```bash
$ npm install
```

<br/>

## 🚀 Running the app

```bash
# development
$ npm run dev

# build
$ npm run build

# production mode
$ npm run start
```

<br/>

## 🔒 Environment configuration

```
NODE_ENV

DB_NAME
DB_PORT
DB_HOST
DB_USERNAME
DB_PASSWORD

JWT_ACCESS_TOKEN_SECRET

TWILIO_ACCOUNT_SID
TWILIO_AUTH_TOKEN
TWILIO_NUMBER
```
