# Music API Server Example

## 📚 Description

This is an example of a music API server with DB connection (Node.js, Express, Sequelize).

<br/>

## 🛠️ Installation

```bash
$ npm install
```

<br/>

## 🚀 Running the app

```bash
# development
$ npm run dev

# build
$ npm run build

# production mode
$ npm run start
```

<br/>

## 🔒 Environment configuration

```
NODE_ENV

DB_USERNAME_DEV
DB_PASSWORD_DEV
DB_NAME_DEV
DB_HOST_DEV
```
